## 项目简介
**北京轨道地图（BSMS）** 是一个基于 Leaflet.js 实现的网页版北京轨道交通可视化工具，支持交互式操作和强大的地铁线路查询与展示功能。项目旨在为用户提供清晰、美观且操作友好的北京地铁线路与换乘查询体验。

## 主要功能
- **地铁线路与站点可视化**：完整还原北京地铁线路，所有车站均可交互、查看站点归属线路。
- **底图透明度与背景色调节**：可自定义底图透明度、设置页面背景色，适应不同浏览、演示场景。
- **线路图例**：每条线路配有图例，点击图例项可高亮对应线路。
- **站点搜索定位**：支持站点名称模糊搜索，输入站名可直接定位并弹窗展示。
- **路径规划**：
  - 输入起点和终点站支持车站补全。
  - 支持“最少站数优先”与“最少换乘优先”两种路径算法，双模式智能切换。
  - 可详细展示换乘方案、途径线路与每站信息。
  - 路径高亮，在地图上突出显示经过线路和关键车站。
  - 一键清除查询结果。
- **显示/隐藏控制**：
  - 可分别设置是否显示站点、站名、图例等。
- **底图服务切换**：支持高德地图（AMap）与 OpenStreetMap（OSM）底图切换。
- **浏览器本地记忆**：所有设置及开关均自动保存，刷新后自动恢复上次状态。
- **界面美观**：自适应窗口，支持移动端浏览，界面友好，交互流畅。

## 数据格式说明

所有线路及站点数据在 HTML 文件中以 JavaScript 变量直接维护，无需请求外部 JSON 文件。核心数据结构如下：

### 站点数据（`allStations`）

```js
const allStations = {
  "古城": { lat: 39.9072, lng: 116.1902 },
  // ... 其它站点
};
```

### 线路与走向（`lineConnections`）

```js
const lineConnections = {
  "1号线": ["古城","八宝山","玉泉路", ...],
  "2号线": [...],
  // 更多线路
};
```

每条线路的数组成员为经停站点名称，顺序代表线路走向。

## 常见问题 (FAQ)
**1. 数据如何与真实线路保持一致？**

地铁线路数据均内嵌在 HTML 文件中，修改和更新数据需编辑 beijing-subway-map-super.html 里的线路和站点数组，暂不支持自动在线同步外部数据源。


**2. 是否支持移动端？**

支持部分移动端浏览，但最佳体验建议使用桌面浏览器。经代码实际测试，某些交互在手机浏览器上体验有限。


**3. 地铁线路怎么高亮？**

在图例或站点弹窗点选线路，可以高亮显示/取消高亮对应线路。


**4. 如何查看站点的详细信息？**

在地图上点击任意一个车站节点，会弹出包含站名和换乘信息的气泡。具体实现可见 beijing-subway-map-super.html 中的 `showStationInfo` 函数。


**5. 如何搜索站点并快速定位？**

在“工具”区域上方“搜索站点”输入框输入名称，可模糊匹配，北京地铁所有站点均可搜，点击搜索结果即可让地图跳转至目标站点，并弹出该站点详情。


**6. 如何路径查询与模式切换？**

输入起点和终点后，点击“查询”按钮（`id="searchRouteBtn"`），下方会展示最优路线。上方有“向左/向右箭头”按钮（`id="toggleRouteModeBtnLeft/Right"`），分别切换“站数优先”或“换乘次数优先”模式，支持不同出行偏好。
